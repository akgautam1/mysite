# My Simple GitHub Pages Site

This is a simple static website template designed to be published on GitHub Pages.

## How to edit
- `index.html` — edit content & sections.
- `style.css` — change colors, fonts, layout.
- `assets/images/hero.jpg` — replace hero image.
- `script.js` — add small interactivity.

## Deployment
Follow the instructions in the root of this repo to publish to GitHub Pages.
